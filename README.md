> 이 프로젝트는 Google Deepmind의 Antigravity AI 코딩 어시스턴트를 테스트하기 위해 제작되었습니다.

## 📖 프로젝트 소개

스크롤에 반응하여 3D 공간에서 별자리를 탐험할 수 있는 인터랙티브 웹페이지입니다. Three.js를 활용하여 구현된 이 프로젝트는 아름다운 별의 반짝임, 별자리 연결선 애니메이션, 그리고 유성 효과를 포함합니다.

## ✨ 주요 기능

- 🌟 **별 반짝임 효과**: 각 별이 독립적으로 다른 속도와 밝기로 반짝입니다
- 🎨 **별 색상 변화**: 별의 온도에 따라 파란색, 흰색, 노란색, 주황색, 붉은색으로 표현
- 🔗 **별자리 연결선**: 각 별자리가 포커스될 때 노드를 따라 순차적으로 그려지는 애니메이션
- 📝 **별자리 정보**: 한글 제목, 설명, 아이콘 이미지와 함께 각 별자리 정보 표시
- 💫 **유성 효과**: 랜덤하게 나타나는 유성이 꼬리를 남기며 날아감
- 📱 **반응형 디자인**: 모바일과 데스크톱 모두 지원

## 🎯 포함된 별자리

## 🚀 실행 방법

### 의존성 설치
```bash
npm install
```

### 개발 서버 실행
```bash
npm run dev
```

서버가 자동으로 http://127.0.0.1:8080 에서 실행되고 브라우저가 열립니다.

## 🛠️ 기술 스택

- **JavaScript (ES6+)** - 메인 로직
- **HTML5 & CSS3** - 마크업 및 스타일링
- **http-server** - 로컬 개발 서버

## 💡 주요 구현 기술

### 1. 카메라 이동 시스템
스크롤 위치에 따라 카메라가 부드럽게 이동하며 각 별자리를 탐험합니다.

### 2. 별 생성 및 렌더링
- Canvas를 사용한 별 텍스처 생성
- Radial gradient로 미세한 글로우 효과
- 온도 기반 색상 변화 (파란색 고온 → 붉은색 저온)

### 3. 별자리 연결선 애니메이션
- BufferGeometry를 사용한 라인 렌더링
- 노드별 순차적 그리기 애니메이션
- 상태 기반 lifecycle 관리

### 4. 유성 효과
- 랜덤 생성 및 소멸 시스템
- 꼬리(trail) 효과 구현
- 페이드아웃 애니메이션

## 🎮 사용 방법

1. 페이지를 스크롤하여 각 별자리로 이동
2. 오른쪽 마커를 클릭하여 특정 별자리로 바로 이동
3. 왼쪽 패널에서 별자리 정보 확인

## 📊 성능 최적화

- 별 크기 고정으로 불필요한 계산 제거
- 유성 lifecycle 관리로 메모리 누수 방지
- requestAnimationFrame 기반 부드러운 애니메이션
